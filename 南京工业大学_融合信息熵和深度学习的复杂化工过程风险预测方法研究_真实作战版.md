## 2. 技术路线

```
TEP 原始数据 (XMEAS1~XMEAS22)
    ↓
小波去噪 (db4 + soft threshold + 3层，工程默认)
    ↓
滑动窗口 (window=150, step=1)
    ↓
Spearman 相关分析 + 阈值 0.2 → 二值邻接矩阵
    ↓
结构熵 E = -Σ(p_i·ln p_i),  p_i = k_i/Σk_i
    ↓
跨工况统一 min-max 归一化 → R_t ∈ [0,1]
    ↓
风险分级 (4档: 低/较低/较高/高)
    ↓
监督样本构造 (lookback=12, 单步回归)
    ↓
LSTM / Bi-LSTM / Attention-Bi-LSTM 训练预测
    ↓
指标评估 (RMSE, ARGE, R²)
    ↓
图表输出 + 结题论文
```

## 8. 快速参考：关键公式

**结构熵（Eq.1）**
```
E = - Σ_{i=1}^{N} (k_i / Σk_i) · ln(k_i / Σk_i)
```

**相对风险序列（Eq.5）**
```
R_t = (E_t - min(E)) / (max(E) - min(E))
# min/max 在三工况全部窗口上取全局值
```

**注意力（Eq.4）**
```
y_i = Σ_j [exp(a·<s_{i-1}, h_j>) / Σ_k exp(a·<s_{i-1}, h_k>)] · h_j
# 加性注意力，s 为全连接层输出
```

**主指标**
```
RMSE = sqrt( mean( (y_true - y_pred)^2 ) )
ARGE = mean( |y_true - y_pred| / y_true )
R²   = 1 - SS_res / SS_tot
```
